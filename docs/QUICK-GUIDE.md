# 엄마 알리미 - 간단 가이드

> 5가지 핵심 기능 빠른 시작

---

## 1. 메시지 작성하기

1. **"+ 메시지 추가"** 버튼 클릭
2. 내용 입력
3. 날짜/시간 선택
4. **"저장"**

---

## 2. 디스플레이에서 보기

1. **홈 → 화면(🖥️) 아이콘** 클릭
2. 화면 터치로 오디오 활성화
3. 메시지가 큰 글씨로 표시됨

---

## 3. 음성으로 듣기

1. 메시지 작성 시 **"음성 읽기"** 체크
2. 디스플레이에서 **"듣기"** 버튼 클릭
3. 또는 설정한 알림 시간에 자동 재생

---

## 4. 반복 메시지 설정

1. 메시지 작성 → **"반복 설정"** 켜기
2. 요일 선택 (월, 수, 금 등)
3. 매주 해당 요일에 자동 표시

---

## 5. 알림 받기

1. **설정 → 알림** 열기
2. **푸시 알림** 활성화
3. 가족이 메시지 작성하면 알림 수신
