## 1. 소개 및 설치

### 1.1 앱 소개

**엄마 알리미**는 가족 간의 메시지를 전달하는 디지털 메시지 보드입니다.

**주요 특징:**
- 태블릿에 메시지를 크게 표시
- 음성으로 메시지를 읽어줌
- 스마트폰에서 메시지 작성/관리
- 실시간 동기화

### 1.2 PWA 설치 방법

#### Android (Chrome)

1. Chrome 브라우저로 앱 접속
2. 주소창 오른쪽 메뉴(⋮) 클릭
3. **"앱 설치"** 또는 **"홈 화면에 추가"** 선택
4. "설치" 확인

#### Android (삼성 인터넷)

1. 삼성 인터넷으로 앱 접속
2. 하단 메뉴(≡) 클릭
3. **"페이지를 앱으로 추가"** 선택
4. "추가" 확인

#### iOS (Safari)

1. Safari로 앱 접속
2. 하단 공유 버튼(□↑) 클릭
3. **"홈 화면에 추가"** 선택
4. "추가" 확인

> **참고**: iOS에서는 푸시 알림이 제한적으로 작동합니다.

## 2. 회원가입 / 로그인

### 2.1 계정 생성

1. 앱 실행 후 **"회원가입"** 탭 선택
2. 정보 입력:
   - **이메일**: 사용할 이메일 주소
   - **비밀번호**: 8자 이상, 영문+숫자 포함
   - **이름**: 실명
   - **닉네임**: 가족에게 보여질 이름 (예: 아버지, 막내)
   - **성별**: 남성/여성 선택
3. **"회원가입"** 버튼 클릭
4. 이메일 인증:
   - 입력한 이메일로 인증 메일 발송됨
   - 메일의 **"인증하기"** 링크 클릭
   - 인증 완료 후 로그인 가능

### 2.2 로그인

1. 이메일과 비밀번호 입력
2. **"로그인"** 버튼 클릭

### 2.3 비밀번호 찾기

1. 로그인 화면에서 **"비밀번호를 잊으셨나요?"** 클릭
2. 가입한 이메일 입력
3. **"재설정 링크 보내기"** 클릭
4. 이메일에서 **"비밀번호 재설정"** 링크 클릭
5. 새 비밀번호 입력 후 저장

### 2.4 이메일 찾기

1. 로그인 화면에서 **"이메일 찾기"** 클릭
2. 가입 시 입력한 전화번호 입력
3. 등록된 이메일 표시

## 3. 가족 만들기 / 참여하기

메시지를 보내려면 먼저 **가족**에 속해야 합니다.

### 3.1 새 가족 생성

1. **설정**(⚙️) → **가족 관리**
2. **"새 가족 만들기"** 클릭
3. 가족 이름 입력 (예: 김씨 가족)
4. **"만들기"** 클릭
5. 생성한 사람이 **관리자**가 됩니다

### 3.2 가족 검색 및 참여 요청

1. **설정** → **가족 관리**
2. **"가족 찾아 참여하기"** 클릭
3. 참여할 가족 이름 검색
4. 찾은 가족 카드에서 **"참여 요청"** 클릭
5. 가족 관리자가 수락하면 참여 완료

### 3.3 참여 요청 수락/거절 (관리자)

1. **설정** → **받은 참여 요청** 확인
2. 요청 카드에서:
   - ✓ : 수락
   - ✗ : 거절
3. 수락 시 해당 사용자가 가족에 추가됨

### 3.4 활성 가족 변경

여러 가족에 속한 경우:

1. **설정** → **내가 속한 가족** 목록
2. 사용할 가족 카드에서 **"선택"** 클릭
3. 선택한 가족이 **활성**으로 표시됨
4. 메시지는 활성 가족에서만 보임

## 4. 메시지 작성 및 관리

### 4.1 새 메시지 작성

1. **홈** 화면에서 **"+ 메시지 추가"** 버튼 클릭
2. 메시지 정보 입력:
   - **내용**: 전달할 메시지 (최대 500자)
   - **표시 날짜**: 메시지를 보여줄 날짜
   - **표시 시간**: 특정 시간 또는 "종일"
   - **중요도**: 일반/중요/긴급
3. TTS 설정 (선택):
   - **음성 읽기** 체크
   - **알림 시간** 추가 (여러 개 가능)
4. **"저장"** 클릭

### 4.2 표시 시간 설정

- **시간 지정**: 특정 시간에 메시지 강조
- **종일**: 하루 종일 표시
- **바로가기 버튼**: -60, -30, -15, -10, -5, 정각, +5, +10분

### 4.3 메시지 수정

1. 메시지 카드의 **연필 아이콘(✏️)** 클릭
2. 내용 수정 후 **"저장"**

> **참고**: 본인이 작성한 메시지만 수정 가능. 가족 관리자는 모든 메시지 수정 가능.

### 4.4 메시지 삭제

1. 메시지 카드의 **휴지통 아이콘(🗑️)** 클릭
2. 확인 후 삭제

### 4.5 달력에서 관리

1. **홈** → **달력 아이콘(📅)** 클릭
2. 달력에서 메시지가 있는 날짜 확인:
   - **파란색 점**: 1-2개 메시지
   - **빨간색 점**: 3개 이상 메시지
3. 날짜 클릭 시 해당 날짜 메시지 목록으로 이동

## 5. 반복 메시지 설정

매일 또는 특정 요일마다 반복되는 메시지를 설정할 수 있습니다.

### 5.1 반복 메시지 생성

1. **새 메시지 작성** 화면에서
2. **"반복 설정"** 토글 활성화
3. 반복 요일 선택:
   - 원형 버튼으로 요일 선택 (복수 선택 가능)
   - 예: 월, 수, 금 선택 → "월 수 금" 반복
4. 반복 이름 입력 (선택): 예) "아침 약 알림"
5. **"저장"**

### 5.2 반복 메시지 관리

1. **홈** → **반복 아이콘(🔁)** 클릭
2. 반복 메시지 목록 표시 (알람 앱 스타일)
3. 각 메시지에서:
   - **토글**: 반복 켜기/끄기
   - **"오늘만 건너뛰기"**: 오늘은 표시 안 함
   - **"삭제"**: 완전 삭제

### 5.3 특정 날짜 건너뛰기

1. 반복 메시지가 표시된 날에
2. 메시지 카드에서 **눈 아이콘(👁️‍🗨️)** 클릭
3. "이 날짜에 숨기기" 선택
4. 해당 날짜만 메시지 숨김

## 6. 디스플레이 화면 사용법

### 6.1 디스플레이 화면 열기

1. **홈** → **화면 아이콘(🖥️)** 클릭
2. 또는 주소창에 `/display` 입력

### 6.2 처음 사용 시

1. **"화면을 터치하세요"** 메시지 표시
2. 화면 아무 곳이나 터치
3. 오디오 활성화 완료 → 알림음/TTS 사용 가능

### 6.3 메시지 확인

- 오늘 표시할 메시지가 목록으로 표시
- **"듣기"** 버튼: 해당 메시지 음성 재생
- 시간이 되면 자동으로 해당 메시지로 스크롤

### 6.4 야간 모드

- 설정된 시간(기본 20:00~06:00)에 자동 진입
- 검은 화면 표시
- 화면 터치 시 즉시 해제

### 6.5 화면 꺼짐 방지

- 디스플레이 화면에서는 자동으로 화면이 꺼지지 않음
- 하단에 **"🔆 화면 유지"** 표시 확인

### 6.6 태블릿 설정 권장

1. **전원 연결** 상태 유지
2. **화면 자동 꺼짐**: 사용 안 함 또는 최대 시간
3. **밝기**: 적절히 조절
4. **방해 금지 모드**: 끄기 (알림 받으려면)

## 7. TTS 및 알림 설정

### 7.1 TTS 음성 선택

1. **설정** → **TTS 설정**
2. **음성 선택** 드롭다운에서 선택:
   - Wavenet 음성: 더 자연스러움
   - Standard 음성: 기본
   - 남성/여성 음성 선택 가능
3. **"테스트"** 버튼으로 미리듣기
4. **저장**

### 7.2 읽기 속도 조절

1. **설정** → **TTS 설정**
2. **읽기 속도** 슬라이더 조절:
   - 0.5: 매우 느림
   - 1.0: 보통
   - 1.5: 빠름
3. **저장**

### 7.3 알림 시간 설정

메시지별로 알림 시간을 설정:

1. **메시지 작성/수정** 화면
2. **"음성 읽기"** 체크
3. **"알림 시간 추가"** 클릭
4. 시간 입력 (예: 09:00)
5. 여러 시간 추가 가능

### 7.4 푸시 알림 설정

1. **설정** → **알림 설정**
2. **"푸시 알림"** 토글 활성화
3. 알림 유형 선택:
   - **새 메시지 알림**: 가족이 메시지 작성 시
   - **가족 참여 요청 알림**: 누군가 가족에 참여 요청 시
4. **저장**

### 7.5 야간 모드 시간 설정

1. **설정** → **디스플레이 설정**
2. **"야간 모드"** 토글 활성화
3. **시작 시간** 설정 (기본: 20:00)
4. **종료 시간** 설정 (기본: 06:00)
5. **저장**

## 8. FAQ / 문제 해결

### Q1. 메시지가 안 보여요

**확인 사항:**
1. 올바른 가족이 **활성** 상태인지 확인
2. 메시지의 **표시 날짜**가 오늘인지 확인
3. 반복 메시지의 경우 오늘 요일이 선택되었는지 확인

### Q2. 음성이 안 나와요

**확인 사항:**
1. 디스플레이 화면에서 **터치로 오디오 활성화** 했는지 확인
2. 기기 **볼륨**이 켜져 있는지 확인
3. 메시지의 **"음성 읽기"** 옵션이 켜져 있는지 확인

### Q3. 푸시 알림이 안 와요

**확인 사항:**
1. **설정 → 알림**에서 푸시 알림 활성화 확인
2. 브라우저/기기 **알림 권한** 허용 확인
3. iOS Safari에서는 푸시 알림이 제한적

### Q4. 야간 모드가 안 꺼져요

**해결:**
- 화면을 **터치**하면 즉시 해제됩니다
- 설정에서 야간 모드 **시간 변경** 가능

### Q5. 화면이 자동으로 꺼져요

**해결:**
1. 기기 **화면 자동 꺼짐** 설정을 "사용 안 함"으로 변경
2. 디스플레이 화면 하단에 **"🔆 화면 유지"** 표시 확인
3. 지원되지 않는 브라우저일 수 있음 (Chrome 권장)

### Q6. 다른 가족 메시지가 보여요

**확인:**
1. **설정 → 가족 관리**에서 올바른 가족이 **활성** 상태인지 확인
2. 활성 가족 변경 후 홈으로 돌아가기

### Q7. 비밀번호를 잊어버렸어요

**해결:**
1. 로그인 화면 → **"비밀번호를 잊으셨나요?"**
2. 이메일 입력 → 재설정 링크 발송
3. 이메일에서 링크 클릭 → 새 비밀번호 설정

### Q8. 회원 탈퇴하고 싶어요

**방법:**
1. **설정 → 계정 관리**
2. **"회원 탈퇴"** 클릭
3. 30일 유예 기간 후 완전 삭제
4. 유예 기간 내 다시 로그인하면 취소 가능
