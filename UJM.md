# 🗺️ 가족 메시지 보드 - 사용자 여정 맵 (User Journey Map)

## 프로젝트 개요
- **목적**: 경도 인지 장애 어르신을 위한 디지털 메시지 보드
- **주 사용자**: 팔순 어머님 (경도 인지 장애)
- **보조 사용자**: 가족 구성원 (10명 미만)
- **주 디바이스**: 갤럭시 탭 7 FE (식탁 옆 고정)
- **입력 디바이스**: 스마트폰 (가족)

## 🎯 핵심 인사이트

### 어머님 입장
1. **수동적 사용**: 알림이 오면 듣기만 함
2. **터치 최소화**: 귀찮아하심 → 음성 중심
3. **큰 글씨 필수**: 메시지 내용 특히 크게
4. **야간 방해 금지**: 20시~06시 조용

### 가족 입장
1. **빠른 등록**: 스마트폰으로 쉽게
2. **상태 확인**: 알림 발송 여부
3. **안심**: 어머니가 메시지 보고 계신지

### 시스템 입장
1. **신뢰성**: 알림 절대 놓치면 안됨
2. **자동화**: 반복 메시지, 시간대별 설정
3. **안정성**: 인터넷 끊겨도 캐시 표시

## 📋 주요 요구사항 요약

### 기기 및 환경
- **태블릿**: 갤럭시 탭 7 FE
- **위치**: 식탁 옆 고정
- **항상 켜놓음**: 자동 켜짐/꺼짐 없음
- **화면 보호**: 20:00~06:00 블랙 스크린

### 음성 기능
- **읽어주는 횟수**: 알림 시 한 번만
- **음성 볼륨**: 시간대별로 다르게
  - 주간(06:00-20:00): 높은 볼륨
  - 야간(20:00-06:00): 음소거

### 알림 전략
- **사전 알림**: 5분 전
- **메인 알림**: 설정 시간
- **재알림**: 10분 후 (확인 안 했을 시)
- **가족 알림**: 불필요 (확인 기능 없음)

### 데이터 관리
- **오래된 메시지**: 2년 후 자동 삭제
- **확인된 메시지**: 계속 표시 (숨기지 않음)
- **반복 메시지**: 시작일, 종료일 등록 가능

### 가족 사용자
- **인원**: 10명 미만
- **주 작성자**: 철수(나)
- **입력 디바이스**: 스마트폰

### 기술 요구사항
- **비용**: 무료 플랜
- **백업**: 불필요
- **재부팅**: 자동 실행
- **인터넷 끊김**: 오프라인 모드 (캐시된 메시지)
- **앱 오류**: 알림 불필요
