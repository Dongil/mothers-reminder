# 알림음 파일

다음 오디오 파일을 추가해주세요:

- `chime.mp3` - 일반 알림용 차임벨 (부드러운 소리)
- `alert.mp3` - 긴급 알림용 경고음 (주의를 끄는 소리)

## 권장 사항

- 형식: MP3 (브라우저 호환성)
- 길이: 1-3초
- 볼륨: 적당한 크기 (어르신이 놀라지 않도록)
- 음색:
  - chime.mp3: 따뜻하고 부드러운 벨 소리
  - alert.mp3: 명확하지만 자극적이지 않은 알림음
